    • Ejemplo de Red
    • IPv4
      • Son identificadores
      • Permiten para identificar los dispositivos que tienen conexiones a la red, si tienen una sola conexion a la red
      • Un dispositivo puede tener una, o varias direcciones IP, dependiendo de la cantidad de conexiones a la red
      • Reglas
        • Son Unicas
        • Son universales
          • Todos los dispositivos tienen que tener direccion IP
        • 32 bits
          • Notacion decimal con puntos
            • 1111 1111 - 1111 1100 - 0000 0001 - 1000 0000
            • 255.252.1.128
        • Tienen 2 partes
          • Red y Host
        • Son dependientes de la localización
        • Son jerarquicas
          • Subredes son pedacitos de una red mas grande
      • La primera ip de una red esta reservada a la red
        • Red R1
          • 10.0.0.0 -> Reservada (All 0) -> Net ID
          • 10.255.255.255 -> Reservada (All 1) -> Net Broadcast
      • Clases de redes
        • A |8|24| -> 2^24 - 2 Redes disponibles
          • Empiezan con 00
          • 127.0.0.1 reservado para Local host / loopback
          • Ejemplo: 10.0.0.0
          • 8 bits fijos para red
        • B |16|16| -> 2^16 - 2 Redes disponibles
          • Empieza con 10
          • Ejemplo: 128.75.0.0
          • 16 bits fijos para red
        • C |24|8| -> 2^8 - 2 Redes disponibles
          • Empiezan con 110
          • Ejemplo: 192.0.2.0
          • 24 bits fijos para red
      • Mascara de subred
        • Cubre una parte de la IP del host
        • Permite creas subredes
        • Se pueden crear en multiplos de 2, (2, 4, 8, 16, 32)
        • Ejemplo:
          • Clase B
            • 132.10.0.0
            • 255.255.0.0 -> Mascara natural ( /16 -> indica cuantos bits se cubren usando una operacion AND)
            • 255.255.192.0 -> Mascara /18 -> 4 subredes
              • 132.10.0.0 -> 132.10.63.255.255
              • 132.10.64.0 -> 132.10.127.255
              • 132.10.128.0 -> 132.10.191.255
              • 132.10.192.0 -> 132.10.255.255
            • 255.255.240.0 -> Mascara /20 -> 16 subredes
              • 132.10.0.0 -> 132.10.15.255
              • 132.10.16.0
              • 132.10.32.0
              • 132.10.48.0
              • 132.10.64.0
              • 132.10.80.0
              • 132.10.96.0
              • ..|..

  • Probabilidad y Estadistica
    • Origen
      • Estadistica proviene de statisticum collegium ("consejo de Estado")
      • Origen relacionado con el gobierno
    • Estadistica
      • Es la ciencia que recolecta, organiza, presenta, analiza e interpreta datos para permitir una mejor toma de decisiones
      • Descriptiva
        • Metodos de organizar, resumir y presentar datos.
      • Inferencial
        • Decision, estimado, prediccion o generalizacion de una poblacion basada en una muestra.
    • Una muestra es una porcion de la poblacion de interes
    • Tipos de variables
      • Cualitativa o atributiva
        • La caracteristica que se estudia es no numérica
      • Cuantitativa
        • La informacion es reportada de manera numerica
        • Discreta
          • Solo asume ciertos valores
        • Continua
          • Puede asumir cualquier valor en un rango
    • Nominal Level Data
      • Observations of a qualitative variable can be classified and counted
      • No particular order
    • Ordinal Level Data
      • Classifications are represented by sets of labels or names that have relative values (high, medium, low)
      • The data classified can be ranked or ordered
    • Interval Level Data
      • Data classifications are otdered according to the amount of charasteristics they possess
      • Equal differences are represented by equal differences in the measurements
    • Ratio Level Data
      • Data classifications are ordered according to the amount of the characteristics they possess
      • Equal differences in the characteristic are represented by equal differences in the numbers assigned to the classifications
      • Zero is the absence of the characteristic and the ratio between two numbers is meaningful
